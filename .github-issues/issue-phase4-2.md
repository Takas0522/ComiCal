## 概要
Cosmos DB のコンテナ初期化とインデックスポリシーを設定するスクリプトを作成

## 対象ファイル
- 新規: `scripts/setup-cosmosdb.ps1` または C# コンソールアプリ
- または `api/Comical.Api/Startup.cs` 内で初期化

## 作業内容
1. Cosmos DB コンテナの作成:
   - `comics` コンテナ（サーバーレスモード推奨）
     - パーティションキー: `/id`
     - インデックスポリシー:
       - 範囲インデックス: `/salesDate`, `/title`, `/author`
       - 除外: `/imageBaseUrl` (存在しない)
   - `config-migrations` コンテナ（サーバーレスモード）
     - パーティションキー: `/id`
2. 初期化スクリプトの作成:
   - Azure認証
   - コンテナ存在チェック
   - 作成処理
3. ドキュメント作成:
   - セットアップ手順を README に追加

## 依存関係
- **前提**: Phase 1-2 完了（Cosmos DB接続設定）
- **後続**: Phase 4-3（設定ファイル更新）

## 完了条件
- [ ] セットアップスクリプトが動作する
- [ ] インデックスポリシーが正しく設定される
